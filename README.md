# Morse Converter
A PCB with case that converts and plays messages in Morse code. I based the RP2040's schematic off the Devboard tutorial by KaiPereira but added many of my own components. It is an easy to set up, standalone converter with a keypad-input alphabet and onboard speaker. I chose this to improve my basic circuitry knowledge and designing PCBs, along with controlling hardware functions from python code. I have always liked using codes and ciphers to communicate, but memorizing the Morse alphabet was difficult. This machine does the conversion for you, only needing an input of plain text.
# Using the Converter
Press a key the number of times to cycle to the chosen letter and click enter to type a letter. Ex: To print "E", press the DEF key 2 times and click enter. Inputting a double ENTER adds a word space and triple ENTER submits the entire message.
